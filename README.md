# Arcade Mania

Ez itt az **Arcade Mania** projekt **Backend** github oldala. 

---

A projekt backend-jével kapcsolatos összes dokumentum ezen a github oldalon található meg. Ezek egész pontosan a következők: API, WPF és a hozzájuk tartozó tesztek. A WPF applikációt a mysetup.exe segítségével lehet telepíteni, ezáltal a telepítás után elegendő csak a webAPI-t futatni a Visual Studio 2022 segítségével, a WPF-et nem kell szükségszerűen onnan indítani.
